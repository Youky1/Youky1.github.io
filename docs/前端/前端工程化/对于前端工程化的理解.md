# 对于前端工程化的理解

## 模块化

从**文件组织的角度**，把大文件拆分为多个互相依赖的小文件。

## 组件化

从**设计的角度**，将UI从大到小进行拆分。

一个页面是一个大的组件 **=>** header、footer、sidebar等等中等组件 **=>** 按钮、输入框等小组件

## 规范化

在项目的规划阶段，制定好统一的规范，包括：
- 目录结构的指定
- 编码规范
- 接口文档的规范
- 组件划分
- git分支
- commit信息格式
- 定期的code review

## 自动化

将重复性工作交由机器自动完成，如：
- 自动化测试
- 自动化部署
